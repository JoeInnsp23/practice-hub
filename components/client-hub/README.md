# Client Hub Components

**Location**: `components/client-hub/`
**Type**: UI Components
**Count**: 13 components

## Components

- `ClientCard.tsx` - Client summary card
- `ClientForm.tsx` - Client creation/edit form
- `ClientList.tsx` - Client listing table
- `CompanyOfficersCard.tsx` - Officers display
- `ComplianceList.tsx` - Compliance tracking list
- `ContactsCard.tsx` - Client contacts card
- `DocumentsCard.tsx` - Document management
- `PSCsCard.tsx` - PSC (Persons with Significant Control) card
- `ServicesList.tsx` - Client services list
- `ClientDetailsTabs.tsx` - Client detail tabs
- `ClientStats.tsx` - Client statistics
- `DirectorsCard.tsx` - Directors card
- `TasksCard.tsx` - Client tasks card

## Usage

All components follow the glass-card design system and use shadcn/ui primitives.
