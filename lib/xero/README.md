# Xero Accounting Integration

**Location**: `lib/xero/`
**Type**: Integration Library
**Status**: Partial

## Overview

Xero accounting API integration for invoice sync and financial data.

## Files

- `client.ts` - Xero OAuth client
- `api-client.ts` - Xero API wrapper
- `sync-service.ts` - Data synchronization
- `webhook-processor.ts` - Webhook event handling
- `token-refresh-worker.ts` - OAuth token refresh
- `middleware.ts` - Credential middleware

## Environment Variables

OAuth-based (no API key required)
