# Google Gemini AI Integration

**Location**: `lib/ai/`
**Type**: Integration Library
**Status**: Active

## Overview

Google Gemini AI integration for document extraction and questionnaire pre-fill.

## Files

- `extract-client-data.ts` - Extract structured data from documents
- `questionnaire-prefill.ts` - AI-powered form pre-filling

## Key Functions

- `extractClientDataFromDocument(file)` - OCR and structured extraction
- `prefillQuestionnaire(context)` - Generate questionnaire responses

## Environment Variables

- `GOOGLE_AI_API_KEY` - Gemini API key

## Documentation

Multi-modal document processing with Gemini 1.5.
